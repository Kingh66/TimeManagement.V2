## ✨ Features

### 📋 Task Organization

* Full CRUD functionality (Create, Read, Update, Delete)
* Keep tasks structured and manageable

### ⏳ Time Tracking

* Log time spent on tasks
* Monitor productivity across activities

### 🖥️ Interactive UI

* Clean, distraction-free interface
* Fast navigation and smooth workflow

### 💾 Data Persistence

* Local storage using SQLite
* Data remains saved between sessions

## 🚀 Getting Started

### ✅ Prerequisites

* Visual Studio 2022 or newer
* ".NET desktop development" workload installed

---

### ⚙️ Installation & Running

#### 1. Clone the repository

```bash
git clone https://github.com/Kingh66/TimeManagement.V2.git
```

#### 2. Open the project

* Navigate to the cloned folder
* Double-click the `.sln` file

#### 3. Build the project

* Go to **Build > Build Solution**
* Or press `Ctrl + Shift + B`
* NuGet packages (e.g., `System.Data.SQLite`) will restore automatically

#### 4. Run the application

* Press `F5`
* Or click **Start**
